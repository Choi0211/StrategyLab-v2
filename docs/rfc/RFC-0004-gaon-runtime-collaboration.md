# RFC-0004: Gaon Runtime Collaboration

Status: Accepted for implementation

This phase adds Gaon Runtime and collaboration contracts on top of Learning Memory and Research Brain.

## Included

- runtime configuration
- event bus
- conversation runtime
- Telegram dry-run adapter
- Notion dry-run mapper/sync
- notification engine
- daily report
- weekly review
- scheduler
- safe dry-run CLI
- Learning Memory claims snapshot and retrieval hardening

## Excluded

- real network calls
- external AI APIs
- vector DB or embeddings
- live trading
- MyMoneyGuard/KIS access
- automatic approvals
