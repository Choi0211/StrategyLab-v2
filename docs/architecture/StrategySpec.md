# StrategySpec

Sprint 107 adds canonical, versioned strategy specifications.

Fields:

- entry rules
- exit rules
- filters
- position sizing
- stop loss
- take profit
- universe
- max positions
- rebalance
- timeframe

The Turtle fixture represents a 20-day breakout, trend filter, volume filter,
5 percent stop, 10-day low exit, and maximum two holdings. StrategySpec is data,
not generated Python code.
