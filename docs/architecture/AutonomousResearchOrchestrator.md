# Autonomous Research Orchestrator

Sprint 100 connects memory search, novelty detection, candidate generation,
validation-style critique, bounded improvement, quality scoring, tournament
ranking, memory save, and report generation.

The orchestrator is fixture-backed and deterministic in this release. It is an
AI Quant Researcher foundation, not an execution engine. It never performs live
orders, automatic Champion promotion, source edits, shell commands, arbitrary
SQL, or deployment changes.

Safe tools added:

- `research_memory_search`
- `strategy_critique`
- `strategy_quality_score`
- `research_candidate_compare`
- `research_lineage`
